# Pixiv Explorer スキル

Pixivのイラスト探索・分析・ダウンロードを総合的に支援するスキルです。

## 実行フロー

### Step 1: ヒアリング

ユーザーに以下を確認してください：

1. **目的**: 何を探していますか？（トレンド確認 / 特定テーマ / 特定作者 / おすすめ）
2. **フィルタ条件**:
   - AI生成作品: 含める / 除外 / AI生成のみ
   - R-18: 含める / 除外（デフォルト: 除外）
3. **数量**: 何件程度必要か（デフォルト: 20件）
4. **ダウンロード**: 気に入った作品をダウンロードしたいか

### Step 2: 探索の実行

目的に応じて以下のパターンで探索を実行します。Agentツールを使って並列実行することで効率化してください。

#### パターンA: トレンド分析
1. `get_trending_tags` でトレンドタグを取得
2. `get_trending` でデイリーランキングを取得（mode: "day"）
3. トレンドタグから気になるタグを `search_by_hashtag` で深掘り

#### パターンB: テーマ別探索
1. ユーザーのテーマに基づいて `search_illusts` で検索
2. 関連タグが見つかれば `search_by_hashtag` で追加検索
3. 良い作品の作者を `get_user_illusts` で掘り下げ

#### パターンC: 作者調査
1. `search_users` で作者を検索
2. `get_user_illusts` で作品一覧を取得
3. 気になる作品を `get_illust_detail` で詳細確認

#### パターンD: おすすめ探索
1. `get_recommended` でレコメンド取得
2. 気に入った作品のタグから `search_by_hashtag` で類似作品を探索

### Step 3: 結果の整理・表示

以下の形式で結果をまとめて表示してください：

```markdown
## 探索結果

### フィルタ条件
- AI生成: [含む/除外/のみ]
- R-18: [含む/除外]

### 発見したイラスト

| # | タイトル | 作者 | タグ（主要3つ） | AI | ブクマ | 閲覧数 | ID |
|---|---------|------|----------------|-----|-------|--------|-----|

### 注目ポイント
- [分析コメント: トレンドの傾向、人気のテーマなど]
```

### Step 4: ダウンロード（任意）

ユーザーがダウンロードを希望した場合：

1. ダウンロード対象のイラストIDリストを確認
2. `batch_download` で一括ダウンロード
3. ダウンロード結果を報告

## 著作権に関する注意

- ダウンロードした作品は個人利用の範囲内で使用してください
- 作品の再配布・商用利用は各作者の利用規約に従ってください
- AI生成フィルタは `illust_ai_type` フィールドに基づきますが、作者が正しく設定していない場合があります

## AI生成フィルタについて

Pixivでは2022年11月から作品投稿時にAI生成かどうかの申告が必須になりました。
- `illust_ai_type = 0`: 非AI（手描き）
- `illust_ai_type = 1`: AI生成
- `illust_ai_type = 2`: AI利用（部分的にAIを使用）

ただし、自己申告制のため100%正確ではない点に注意が必要です。
