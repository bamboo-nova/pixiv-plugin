# Pixiv API リファレンスガイド

## ランキングモード一覧

### 通常ランキング
| モード | 説明 |
|--------|------|
| `day` | デイリーランキング |
| `week` | ウィークリーランキング |
| `month` | マンスリーランキング |
| `day_male` | 男性に人気 |
| `day_female` | 女性に人気 |
| `week_original` | オリジナル作品 |
| `week_rookie` | ルーキー作品 |

### R-18ランキング
| モード | 説明 |
|--------|------|
| `day_r18` | R-18デイリー |
| `day_male_r18` | R-18男性向け |
| `day_female_r18` | R-18女性向け |
| `week_r18` | R-18ウィークリー |
| `week_r18g` | R-18Gウィークリー |

## 検索パラメータ

### search_target（検索方式）
- `partial_match_for_tags`: タグ部分一致（デフォルト、最も汎用的）
- `exact_match_for_tags`: タグ完全一致（ハッシュタグ検索向け）
- `title_and_caption`: タイトルとキャプションから検索

### sort（ソート順）
- `date_desc`: 新しい順（デフォルト）
- `date_asc`: 古い順
- `popular_desc`: 人気順（Pixivプレミアム限定）

### duration（期間フィルタ）
- `within_last_day`: 24時間以内
- `within_last_week`: 1週間以内
- `within_last_month`: 1ヶ月以内

## AI生成タイプ

| 値 | 意味 | 説明 |
|----|------|------|
| 0 | 非AI | 手描き作品 |
| 1 | AI生成 | AIで生成された作品 |
| 2 | AI利用 | 部分的にAIを使用した作品 |

### AI生成フィルタの信頼性
- 2022年11月以降、投稿時にAI使用の申告が必須化
- ただし自己申告制のため、一部の作品で正確でない可能性あり
- `exclude_ai=True` で非AI作品のみ、`only_ai=True` でAI作品のみに絞り込み可能
